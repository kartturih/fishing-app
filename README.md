# Fishing App

> An offline-first mobile fishing companion built with Flutter.

## Status

🚧 Early development

The project is currently in the planning and architecture phase.

## Vision

Fishing App helps anglers make better decisions before, during and after a fishing trip by combining:

- Interactive maps
- Personal fishing history
- Catch logging
- Lure management
- Environmental data
- Smart recommendations (future)

## Planned Technology Stack

- Flutter
- Dart
- Riverpod
- Drift (SQLite)
- MapLibre
- Supabase (later)

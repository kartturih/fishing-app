# Project Charter

## Project Name

Fishing App (working title)

---

## Vision

Build a mobile application that helps anglers make better decisions before, during, and after a fishing trip by combining maps, personal fishing history, and environmental conditions into one simple application.

---

## Target Audience

### Initial target

Personal use.

The application is designed around the needs of its creator and will evolve through real-world fishing experience.

### Future target

Recreational anglers looking for a simple, data-driven fishing companion.

---

## Problem Statement

Many anglers struggle with questions such as:

- Where should I fish today?
- Which lure should I use?
- What has worked in similar conditions before?
- How can I keep track of my best fishing spots?

Existing applications often focus on recording catches after a trip rather than helping anglers make better decisions before they start fishing.

---

## Project Goals

The first goal is to build a reliable offline-first fishing application.

The long-term goal is to create a smart fishing companion that learns from the user's own fishing history and provides personalized recommendations.

---

## MVP (Version 0.1)

The first working version will include:

- Interactive map
- GPS location
- Fishing spot management
- Catch logging
- Lure management
- Offline storage

---

## Out of Scope (for MVP)

The following features will NOT be included in the first version:

- User accounts
- Cloud synchronization
- Community features
- AI recommendations
- Garmin integration
- Weather-based recommendations

These features may be added in future versions.

---

## Success Criteria

Version 0.1 is considered successful when the application is stable enough to be used during real fishing trips.
